# BasketPass Landing Page

Landing page estática para **basketpass.cl**.

## Archivos

- `index.html`: página principal.
- `css/styles.css`: estilos de la landing.
- `images/modern_saas_dashboard_with_statistics_and_charts.png`: imagen del dashboard de BasketPass.

## Cómo usar

Abre `index.html` en el navegador o sube estos archivos a GitHub Pages, Vercel, Netlify o cualquier hosting estático.

## Publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube todos los archivos de esta carpeta.
3. En Settings → Pages, selecciona la rama `main` y carpeta `/root`.
4. Guarda y espera la URL pública.

## Pendientes recomendados

- Reemplazar el logo CSS por SVG oficial cuando esté disponible.
- Conectar el botón “Solicitar demo” a un formulario real o CRM.
- Agregar Google Analytics / Meta Pixel si se usará para campañas.
- Crear páginas legales: términos y privacidad.
